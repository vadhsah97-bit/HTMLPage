# Personal Profile Website 🌐

A clean and responsive personal profile website built using HTML and CSS. This project showcases who I am, what I do, and what I’m working toward, all in a simple, visually appealing format.

---

## 📌 Overview

This website serves as my digital introduction. It highlights my background, interests, and skills while keeping the design minimal and easy to navigate. The goal was to create something that feels personal, readable, and smooth across different devices.

---

## 🚀 Features

* Responsive design for desktop and mobile
* Clean and modern UI
* Structured sections for easy navigation
* Custom styling using pure CSS
* Lightweight and fast loading

---

## 🛠️ Built With

* HTML5
* CSS3
